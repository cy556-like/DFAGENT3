# 通用缺陷模板（generic-defect）

## 适用场景

本模板为兜底模板，适用于无法明确归类到涂装/装配/焊接/尺寸的其他缺陷场景。覆盖产品包括但不限于：

- 紧固件
- 橡胶件
- 电子件
- 线束
- 其他未分类产品
- 跨多个类别的复合缺陷

## 覆盖的缺陷类型

| 缺陷类型 | 典型表现 |
|---|---|
| 其他 | 无法归入其他4个模板的缺陷 |
| 未分类 | 用户描述模糊，需要进一步分析 |
| 功能不良 | 产品功能不达标 |
| 性能不达标 | 性能参数低于规格 |
| 外观异常 | 非漆面的外观问题 |
| 可靠性失效 | 寿命/可靠性测试不合格 |

## 预填的 5Why 方向

本模板预填的 5Why 路径为通用逻辑，需根据实际缺陷进一步细化：

1. **第一层**：生产过程的某环节偏离工艺基准，或来料/设备/方法存在异常
2. **第二层**：设备参数偏离、来料变异、操作方法不符合SOP、测量标准不一致
3. **第三层**：设备维护不到位、供应商管控缺失、SOP未量化、检具校准失效
4. **第四层**：设备维护SOP/供应商管理流程/工艺文件不完善
5. **根因**：生产过程关键参数和来料质量缺少量化管控标准、SPC实时监控和供应商深度管理

## 预填的 6M 方向

- **Man**：操作工培训不足、新员工未取得上岗资质、对SOP理解偏差
- **Machine**：设备维护不到位、关键传感器/执行器状态异常
- **Material**：来料批次变异、供应商管控缺失、原料性能波动
- **Method**：SOP未量化、首末件检验缺失、SPC监控未建立
- **Measurement**：检具/测量设备校准过期、与客户测量基准不一致（待确认）
- **Environment**：车间温湿度异常、洁净度不足（待确认）

## 何时选本模板

满足以下任一条件时使用本模板：

1. 用户描述的缺陷无法明确归类到涂装/装配/焊接/尺寸
2. 用户描述的产品类别不在前4个模板的 `product_categories` 中
3. 用户描述模糊，仅说"质量有问题""不合格"等
4. 缺陷涉及多个工艺（如既涉及注塑又涉及装配）

## 使用本模板时的建议

1. **主动追问用户**：使用本模板通常意味着用户描述不够详细，应通过 `AskUserQuestion` 追问：
   - 缺陷的具体表现是什么？
   - 在哪个工序/客户哪个环节发现？
   - 是否有量化数据（不良率、超差多少 mm 等）？
2. **细化 5Why**：通用 5Why 路径仅供参考，应根据实际缺陷重新推演
3. **细化 6M**：根据产品类别，针对性补充 6M 中的具体排查项

## 注意事项

1. 本模板为兜底，应优先尝试匹配前4个模板
2. 若用户后续补充了更具体的缺陷信息，应重新评估是否切换到更精确的模板
3. 本模板的预填内容较为通用，生成的报告需要质量工程师进行更深入的二次编辑
