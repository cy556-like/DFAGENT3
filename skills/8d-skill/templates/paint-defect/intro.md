# 涂装/漆面缺陷模板（paint-defect）

## 适用场景

本模板适用于汽车零部件涂装/喷漆工序产生的各类外观缺陷。覆盖产品包括但不限于：

- 保险杠（前/后）
- 车身面板（门外板、翼子板、引擎盖）
- 后视镜壳体
- 门把手
- 各类外饰涂装件

## 覆盖的缺陷类型

| 缺陷类型 | 典型表现 | 客户反馈示例 |
|---|---|---|
| 颗粒 | 漆膜表面凸起颗粒状异物 | "漆面有黑色颗粒，客户Audit扣分" |
| 流挂 | 漆膜局部厚度过大形成流痕 | "侧围流挂明显，整车外观不达标" |
| 色差 | 与标准色板/相邻件颜色不一致 | "色差ΔE>1.5，客户拒绝接收" |
| 橘皮 | 漆膜表面呈橘皮状纹理 | "橘皮明显，光泽度不达标" |
| 缩孔 | 漆膜表面出现凹坑 | "缩孔密集，需返工" |
| 起泡 | 漆膜与底材间出现气泡 | "烘烤后起泡，附着力失效" |
| 划伤 | 漆膜表面机械划痕 | "运输划伤，客户投诉" |
| 杂质 | 漆膜内/外有异物混入 | "纤维杂质，需排查" |
| 针孔 | 漆膜表面针状小孔 | "针孔严重，需重新喷涂" |

## 预填的 5Why 方向

本模板针对漆面缺陷，预填的 5Why 路径核心指向：

1. **污染物来源**：喷漆室送风过滤器破损、压缩空气含水/油
2. **来料问题**：油漆细度/粘度批次变异
3. **工艺异常**：烘房温度曲线异常
4. **管理根因**：维护保养SOP/来料检验标准缺少量化判定标准

## 预填的 6M 方向

- **Man**：操作工培训不足、点检意识薄弱
- **Machine**：喷漆室/烘房/空压机设备维护不到位
- **Material**：油漆来料细度/粘度批次变异
- **Method**：SOP缺少量化检查标准
- **Measurement**：外观检验标准与客户一致（暂排除）
- **Environment**：车间洁净度/温湿度异常（待确认）

## 何时选本模板

用户描述中出现以下任一关键字，且产品属于外饰涂装件时，首选本模板：

- 漆面、涂装、喷涂
- 颗粒、流挂、色差、橘皮、缩孔、起泡、针孔
- 外观不良（特指漆膜相关）

## 适用标准参考

- **AIAG CQI-12 (Special Process: Coating System Assessment, 2nd Ed., 2019)**：涂装特殊过程系统评估，含前处理、喷涂、烘烤过程的工艺参数控制。本模板 D4 排查方向参照 CQI-12 过程表（Job Audit）。
- **ISO 11997-1 / ASTM B117**：盐雾试验与循环腐蚀测试，D6 验证防腐性能时引用。
- **ASTM D3359 (百格法附着力测试)**：附着力等级（0–5B），D2 问题描述与 D6 验证依据。
- **ASTM E1349 / DIN 6173**：色差测量（ΔE*ab），D2 量化指标依据。
- **VDA 6.3 P6 元素**：制造过程审核中的涂装工序专项。

## 注意事项

1. 若缺陷涉及"颜色"且客户提到"色差ΔE"等量化指标，应保留色差仪测量数据备查
2. 若缺陷涉及"附着力"，应同时启动附着力测试（百格/胶带法）数据收集
3. 涂装缺陷通常客户Audit评分影响较大，D3遏制措施应充分
