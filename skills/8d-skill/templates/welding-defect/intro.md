# 焊接缺陷模板（welding-defect）

## 适用场景

本模板适用于汽车金属零部件焊接工序产生的各类焊接质量问题。覆盖产品包括但不限于：

- 金属支架
- 排气管
- 车身焊接件
- 电池托盘
- 副车架
- 控制臂
- 座椅骨架

## 覆盖的缺陷类型

| 缺陷类型 | 典型表现 | 客户反馈示例 |
|---|---|---|
| 虚焊 | 焊点/焊缝未真正熔合 | "强度测试虚焊脱落" |
| 焊穿 | 焊接热输入过大烧穿母材 | "焊穿漏水，客户退货" |
| 焊渣 | 焊缝表面残留焊渣 | "焊渣飞溅，外观不达标" |
| 强度不足 | 焊接接头强度低于规格 | "拉力测试强度不足" |
| 裂纹 | 焊缝或热影响区出现裂纹 | "X光探伤发现裂纹" |
| 气孔 | 焊缝内部/表面出现气孔 | "超声波探伤发现密集气孔" |
| 咬边 | 焊缝边缘母材被熔化形成凹陷 | "咬边超标，疲劳寿命下降" |
| 未熔合 | 焊缝金属与母材未完全熔合 | "未熔合，强度测试不合格" |
| 焊点偏移 | 焊点位置偏离设计位置 | "焊点偏移，定位失效" |

## 预填的 5Why 方向

本模板针对焊接缺陷，预填的 5Why 路径核心指向：

1. **热输入问题**：焊接电流/电压/速度波动超出工艺窗口
2. **保护气问题**：保护气流量不足或纯度不达标
3. **焊前处理问题**：焊丝/母材表面有油污、锈蚀、氧化皮
4. **管理根因**：焊接关键工艺参数和焊前清洗质量缺少量化管控标准和实时监控

## 预填的 6M 方向

- **Man**：焊工资质过期/未取得对应项目资质、新员工操作不熟练
- **Machine**：焊接电源/机器人/气体流量计维护不到位、轨迹精度漂移
- **Material**：焊丝批次质量变异、母材表面状态异常、保护气纯度不达标
- **Method**：焊接参数SOP缺少量化窗口、焊前清洗SOP缺失
- **Measurement**：焊接强度测试/探伤设备校准过期（待确认）
- **Environment**：车间湿度大导致母材吸潮、焊接区通风异常（待确认）

## 何时选本模板

用户描述中出现以下任一关键字，且产品为金属焊接件时，首选本模板：

- 焊接、焊点、焊缝、焊穿
- 虚焊、未熔合、气孔、咬边、裂纹
- 强度（特指焊接强度）
- 探伤（X光/超声波）

## 适用标准参考

- **AIAG CQI-15 (Special Process: Welding System Assessment, 2nd Ed., 2010)**：焊接特殊过程系统评估，含作业准备、参数监控、设备验证、人员资质。本模板 D4 排查方向、D5 控制项应参照 CQI-15 Table B（过程表）的检查项。
- **AWS D17.2 / ISO 15614-1**：焊接工艺评定（WPQR/PQR）依据，D4 验证焊接参数窗口稳定性时引用。
- **ISO 5817 (B 级)**：焊接缺陷验收等级（气孔/咬边/未熔合分级），D2 问题严重度判定时参考。
- **IATF 16949 §8.5.1.1 控制计划**：焊接参数应纳入控制计划，D5/D7 需更新控制计划。

## 注意事项

1. 焊接缺陷多数涉及安全件，必须启动产品追溯（VIN/批次号），通知客户安全部门
2. 若缺陷涉及"裂纹"或"强度不足"，应立即评估是否需要召回已装车件
3. 焊接缺陷通常需要100%在线探伤+定期强度抽测，D3遏制措施应充分
4. 焊工资质是 IATF 16949 重点审核项，应在 D5-D6 中明确资质复审要求
