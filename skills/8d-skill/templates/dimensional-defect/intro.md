# 尺寸超差模板（dimensional-defect）

## 适用场景

本模板适用于汽车零部件单件尺寸超出规格的各类问题。覆盖产品包括但不限于：

- 注塑件（保险杠、内饰骨架、连接器）
- 冲压件
- 压铸件
- 机加工件
- 各类需要CMM测量的精密件

## 覆盖的缺陷类型

| 缺陷类型 | 典型表现 | 客户反馈示例 |
|---|---|---|
| 关键尺寸超差 | 关键尺寸超出规格上下限 | "CMM测量超差0.5mm" |
| CPK不足 | 过程能力指数CPK<1.33 | "CPK=0.9，PPAP退回" |
| 变形 | 成型后/冷却后形状变化 | "翘曲变形，装配困难" |
| 收缩 | 注塑收缩率不稳定 | "收缩率变异，尺寸波动" |
| 翘曲 | 平面件翘曲超差 | "翘曲超差，平面度不合格" |
| 平面度超差 | 平面度超出规格 | "平面度超差，密封失效" |
| 同轴度超差 | 同轴度超出规格 | "同轴度超差，配合失效" |
| 位置度超差 | 位置度超出规格 | "孔位偏移，装配干涉" |

## 预填的 5Why 方向

本模板针对尺寸超差，预填的 5Why 路径核心指向：

1. **工艺参数问题**：注塑保压/冷却参数偏离工艺窗口
2. **模具/检具问题**：模具型芯磨损、检具校准失效
3. **原料问题**：原料批次变异导致收缩率变化
4. **管理根因**：关键工艺参数（注塑保压/模具定位/检具校准）缺少量化管控标准、SPC实时监控和生命周期管理

## 预填的 6M 方向

- **Man**：操作工装模/调试经验不足、首末件检验执行不到位
- **Machine**：注塑机/冲床/机加工设备参数稳定性差、模具定位精度下降
- **Material**：原料批次变异、回料比例超标、原料含水率超标
- **Method**：工艺SOP缺少量化参数窗口、首末件检验缺失、SPC监控未建立
- **Measurement**：检具/三坐标校准过期、检具与客户CMM基准不一致（待确认）
- **Environment**：车间温度变化大导致模具/原料温度波动（待确认）

## 何时选本模板

用户描述中出现以下任一关键字，且产品为成型/机加工件时，首选本模板：

- 尺寸、超差、CPK、公差
- 变形、收缩、翘曲
- 平面度、同轴度、位置度
- CMM、三坐标、检具

## 与 assembly-defect 的区别

详见 `assembly-defect/intro.md` 中的对照表。简要原则：

- 单件尺寸即超差 → 本模板
- 装配后才体现的问题 → assembly-defect

## 适用标准参考

- **AIAG PPAP 4th Ed.**：D2 涉及 PPAP 退回应明确退回等级（Level 1–5）；D6 验证后重新提交 PSW。
- **AIAG SPC 2nd Ed.**：D4 关键尺寸 SPC 分析，Cpk ≥ 1.33（一般）/ ≥ 1.67（安全件）；子组数 ≥ 25。
- **AIAG MSA 4th Ed.**：D4 测量系统分析，GRR% ≤ 10% 接受，10%–30% 视重要性，> 30% 不可接受。
- **ISO 1101 (GPS 几何产品规范)**：尺寸/形位公差定义与标注。
- **ISO 2768 (一般公差)**：未注线性/角度公差等级（m/c/v 级）。
- **VDA 6.3 P6.2**：制造过程审核中的"作业指导"和"调整/设置"维度。

## 注意事项

1. 若涉及PPAP退回，应同时启动PPAP Level 3 提交资料的复核
2. 若缺陷涉及"CPK不足"，应收集至少25个子组（每组5件）数据用于SPC分析
3. 若缺陷涉及检具，应立即与客户CMM建立比对，确认测量系统一致性
4. 注塑件尺寸问题常与原料含水率相关，应在 D4 中检查原料干燥工艺
