# 装配缺陷模板（assembly-defect）

## 适用场景

本模板适用于汽车零部件装配工序产生的各类装配质量问题。覆盖产品包括但不限于：

- 内饰件（仪表板、门板、立柱护板）
- 外饰装配件（保险杠装配件、格栅）
- 座椅
- 踏板总成
- 各类需要卡扣/螺栓/焊接装配的部件

## 覆盖的缺陷类型

| 缺陷类型 | 典型表现 | 客户反馈示例 |
|---|---|---|
| 间隙超差 | 装配后两件间隙大于/小于规格 | "间隙超差2mm，客户Audit扣分" |
| 面差超差 | 装配后两件表面高度差超规格 | "面差超差，整车外观不达标" |
| 卡扣不到位 | 卡扣未完全卡入 | "卡扣松动，异响" |
| 异响 | 装配后行驶过程产生异响 | "客户行驶异响投诉" |
| 松动 | 装配件未紧固到位 | "松动导致功能失效" |
| 错装 | 装错件或装反方向 | "错装左/右件，客户停线" |
| 漏装 | 缺少紧固件/卡扣 | "漏装卡扣，客户Audit扣分" |
| 力矩不足 | 螺栓/螺母力矩未达规格 | "力矩不足，安全件失效风险" |

## 预填的 5Why 方向

本模板针对装配缺陷，预填的 5Why 路径核心指向：

1. **定位/夹紧问题**：装配夹具定位销磨损、卡扣设计强度不足
2. **力矩控制问题**：操作工未使用扭力扳手、SOP未规定力矩值
3. **来件尺寸问题**：前工序注塑/冲压件尺寸管控缺失
4. **管理根因**：装配过程关键工艺参数（定位/力矩/顺序）缺少量化管控标准和工序间尺寸传递机制

## 预填的 6M 方向

- **Man**：操作工装配经验不足、新员工未取得上岗资质
- **Machine**：装配夹具/扭力工具/定位销磨损或校准失效
- **Material**：来件尺寸批次变异、卡扣/紧固件批次质量波动
- **Method**：SOP未规定量化力矩和装配顺序、首末件检验缺失
- **Measurement**：检具/塞尺/力矩扳手校准过期（待确认）
- **Environment**：装配车间温湿度异常导致塑料件尺寸变化（待确认）

## 何时选本模板

用户描述中出现以下任一关键字，且产品涉及装配工序时，首选本模板：

- 装配、卡扣、力矩、螺栓
- 间隙、面差、错装、漏装
- 异响、松动

## 与 dimensional-defect 的区别

装配缺陷与尺寸超差容易混淆，区分原则：

| 用户描述特征 | 选择模板 |
|---|---|
| 强调"装配后"出现，装配前单件尺寸合格 | assembly-defect |
| 强调单件尺寸即超差，无需装配即体现 | dimensional-defect |
| 强调卡扣/力矩/装配顺序问题 | assembly-defect |
| 强调CPK不足/收缩率/模具问题 | dimensional-defect |

## 适用标准参考

- **AIAG CQI-14 (Automotive Warranty Management)**：D7 横向展开和经验教训库依据。
- **ISO 898-1 / ISO 16048**：紧固件机械性能与力矩规范，D4 排查力矩问题的依据。
- **VDA 6.3 P6.4**：制造过程审核中的"作业/执行"维度（员工操作、工装使用、力矩控制）。
- **AIAG-VDA PFMEA（2019）第 4 步「失效分析」**：装配失效链（功能→失效模式→失效后果）的根因推导框架。
- **IATF 16949 §8.5.1.2 标准化作业**：装配 SOP 必须可视化、可执行；D5/D6 CA 措施应回写 SOP。

## 注意事项

1. 若缺陷涉及安全件力矩，应启动产品追溯（VIN/批次号）并通知客户安全部门
2. 若缺陷涉及"异响"，应安排实车路试复现，记录异响出现的工况
3. 装配缺陷客户停线风险较高，D3遏制措施应充分
